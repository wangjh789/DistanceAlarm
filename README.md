# DistanceAlarm

Client : Flutter Application

Server : Arduino uno (+ ESP01)

## 개발 동기

편의점 알바 중 적외선 센서가 고장나서 뒤쪽의 창고에서 물건 채우고있을 땐 도둑이 털어가도 모름.

매우 불편한데 고쳐줄 생각이 없어보임. 

그래서 초음파 거리센서를 이용해 만들어 보기로 함.
